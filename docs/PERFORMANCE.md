# 人事考核 仕様書

## 1. 評価種別 (Review Type)

| 種別 | 対象 | 周期 | 主用途 |
|---|---|---|---|
| `probation` 試用期間評価 | 新規入社者 | 試用期間終了2週間前〜終了日 | 本採用判定 |
| `mid_year` 半期評価 | 在籍社員 | 9月末 (上期) | 業績フィードバック・目標見直し |
| `annual` 年度評価 | 在籍社員 | 3月末 (下期+通年) | 昇給・賞与・ランク確定 |
| `promotion` 昇格評価 | 推薦者のみ | 随時 | 役職・等級変更 |
| `special` 特別評価 | 個別事案 | 随時 | 異動・降格・特別表彰 |

## 2. 評価ワークフロー (8段階)

```
 ┌─────────────────────────────────────────────────────────────────────┐
 │   1.目標設定        (期初: 上司↔本人で SMART目標を合意)                  │
 │       │                                                              │
 │       ▼                                                              │
 │   2.中間面談        (期央: 進捗確認・目標調整)                          │
 │       │                                                              │
 │       ▼                                                              │
 │   3.自己評価        (期末-2週: 本人が達成度・所見を記入)                 │
 │       │                                                              │
 │       ▼                                                              │
 │   4.一次評価        (期末-1週: 直属上司が評価・スコアを入力)             │
 │       │                                                              │
 │       ▼                                                              │
 │   5.二次評価        (期末: 部門長/校長が確認・調整)                     │
 │       │                                                              │
 │       ▼                                                              │
 │   6.評価会議        (HR主催: 部門間の正規分布調整・キャリブレーション)  │
 │       │                                                              │
 │       ▼                                                              │
 │   7.フィードバック面談  (上司↔本人で確定評価を伝達・次期目標確認)        │
 │       │                                                              │
 │       ▼                                                              │
 │   8.確定           (HR/校長が最終承認 → 給与改定/昇格反映)             │
 └─────────────────────────────────────────────────────────────────────┘
```

### 状態遷移ルール

| 現状態 → 次状態 | 実行可能ロール | 必須条件 |
|---|---|---|
| `draft` → `goal_setting` | 上司, HR | 期間/評価者が設定済 |
| `goal_setting` → `mid_review` | 上司 | 全目標が承認済 (本人+上司) |
| `mid_review` → `self_eval` | 上司 | 中間面談メモ記入済 |
| `self_eval` → `first_eval` | 本人 | 全項目に自己採点+所見 |
| `first_eval` → `second_eval` | 上司 | 全項目スコア+総合所見 |
| `second_eval` → `calibration` | 二次評価者 | 二次コメント記入済 |
| `calibration` → `feedback` | HR | キャリブレーション完了 |
| `feedback` → `finalized` | 上司 | 面談実施日記入 |
| 任意 → `cancelled` | HR | 中止理由必須 |

各遷移時に `audit_logs` に `review.transition` を記録。

## 3. 評価項目 (Evaluation Columns)

### カテゴリ・配点 (デフォルト)

| カテゴリ | ウェイト | 評価項目数 | 評価方法 |
|---|---|---|---|
| `performance` 業績 (MBO) | 60% | 3〜5 目標 | 達成度 (%) → スコア変換 |
| `competency` 能力 | 30% | 4 項目 (固定) | 5段階 |
| `behavior` 行動・価値観 | 10% | 3 項目 (固定) | 5段階 |

### 業績項目 (MBO 目標)

各目標は以下を持つ:
- `title` 目標名
- `description` 詳細・達成基準 (SMART)
- `weight` 配点 (合計100%)
- `target` 目標値 (定量/定性)
- `self_actual` 自己評価の達成度 (%)
- `self_comment` 自己コメント
- `mgr_actual` 上司評価の達成度 (%)
- `mgr_comment` 上司コメント
- `final_score` 最終スコア (1-5)

### 能力項目 (4 固定)

| キー | 名前 | 観点 |
|---|---|---|
| `expertise` | 専門スキル | 自分の業務領域での技術・知識・経験の活用度 |
| `leadership` | リーダーシップ | チーム/後輩への影響力・育成力 |
| `communication` | コミュニケーション | 報連相・会議・対外折衝の質 |
| `problem_solving` | 問題解決力 | 課題発見・分析・実行・振り返り |

### 行動項目 (3 固定)

| キー | 名前 | 観点 |
|---|---|---|
| `teamwork` | 協調性・チームワーク | 部門内/横断連携への貢献 |
| `initiative` | 主体性・挑戦 | 自発的な改善提案・新領域への挑戦 |
| `ethics` | 倫理観・コンプライアンス | 規律遵守・誠実な業務態度 |

### スコア (5段階)

| スコア | ラベル | 定義 |
|---|---|---|
| 5 | S | 期待を大きく超える成果 (上位10%目安) |
| 4 | A | 期待を上回る成果 (次点20%目安) |
| 3 | B | 期待通りの成果 (中位40%目安) |
| 2 | C | 期待を下回る (下位20%目安) |
| 1 | D | 大幅に下回る (下位10%目安) |

## 4. 総合ランク算出

```
 総合スコア = Σ (項目スコア × カテゴリ内ウェイト × カテゴリ間ウェイト)
            = (業績平均 × 0.60) + (能力平均 × 0.30) + (行動平均 × 0.10)

 総合ランク (キャリブレーション前):
   ≥ 4.5 → S
   ≥ 4.0 → A+
   ≥ 3.5 → A
   ≥ 2.5 → B
   ≥ 1.5 → C
   <  1.5 → D
```

評価会議で部門単位の正規分布調整を行い `calibrated_rank` を確定。

## 5. 反映先

| ランク | 標準昇給率 (年度評価) | 賞与係数 |
|---|---|---|
| S | +10% | 2.0× |
| A+ | +7%  | 1.5× |
| A | +5%  | 1.2× |
| B | +3%  | 1.0× |
| C | 0%   | 0.7× |
| D | -2%  | 0.0× |

※ 試用期間評価は別ルール (本採用 / 期間延長 / 不採用)。
