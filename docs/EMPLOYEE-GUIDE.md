# 社員向け 利用ガイド

対象: 一般社員

## 1. ログインする

1. 管理者からメールで届く「ログインID」と「初期パスワード」を確認
2. ログインURL: https://YOUR-DOMAIN/login
3. 初回ログイン後、必ずパスワードを変更してください
4. 管理者から指示があれば 2要素認証 (2FA) を `設定 / 2FA` から設定

## 2. 自分のプロフィールを確認

`組織管理 / 社員一覧` から自分の名前をクリック → 詳細ページ
- 入社日 / 試用期間 / 在留カード期限などが確認可能
- 情報に誤りがあれば 直接編集はできません — HR担当者へ連絡

## 3. 評価フロー — 自己評価の入力

評価期間 (上期9月末 / 下期3月末) になると `評価管理 / 評価カルテ` 自分の名前 → 進行中の評価が表示

### 自分の入力タイミング
- 期初: 上司と1on1で目標を合意 (システムには上司が登録)
- 期末2週前: **自己評価中** ステータスになる
- 各項目に スコア (1-5) と 所見 を入力

| スコア | 意味 |
|---|---|
| 5 (S) | 期待を大きく超える成果 |
| 4 (A) | 期待を上回る成果 |
| 3 (B) | 期待通りの成果 |
| 2 (C) | 期待を下回る |
| 1 (D) | 大幅に下回る |

### 評価項目
| カテゴリ | 配点 | 内容 |
|---|---|---|
| 業績 (MBO) | 60% | 期初に上司と合意した目標の達成度 |
| 能力 | 30% | 専門スキル / リーダーシップ / コミュニケーション / 問題解決 |
| 行動・価値観 | 10% | 協調性 / 主体性 / 倫理観 |

入力完了後 「次のステージへ」 ボタンで提出 → 上司レビューに進む

## 4. 内定者ポータル (新規入社者のみ)

採用通知メールに含まれる招待URLからアクセス:
1. 在留カード / マイナンバー / 給与振込先 等の必要書類をアップロード
2. 担当HRが確認 → 全書類「完了」になれば入社準備完了
3. 不備があれば「差戻し」通知が届くので再提出
4. 招待リンクは 7日有効 — 期限切れの場合は HR に再発行依頼

## 5. プライバシーと個人情報

- マイナンバー / 銀行口座 / パスポート番号 は AES-256 で暗号化保管 (権限を持つHRのみ閲覧可)
- 閲覧/復号操作はすべて監査ログに記録 (改ざん不可)
- 退職5年経過で 上記PIIは自動消去 (法定保持期間)
- 開示請求 / 削除請求 / 訂正請求は HR 窓口へ

## 6. 困ったとき

| 症状 | 連絡先 |
|---|---|
| ログインできない (パスワード忘れ) | HR担当者 |
| 自分の情報が間違っている | HR担当者 |
| アクセス権限がありませんと表示 | システム管理者 |
| 動作が遅い / 表示が崩れる | 別ブラウザを試す → 改善しなければ HR担当者 |
| 不審なメール / セキュリティ事案 | security@your-domain.jp |
